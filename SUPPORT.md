# Getting Support for wp-react

Thank you for using `wp-react`! We want to help you get the most out of this library. Please review the options below to find the best way to get support.

## 📚 Documentation
The best place to start is the official [README.md](README.md). It includes:
* Quick Start and Installation guides
* API References for all Hooks, Components, and Providers
* Examples for Authentication and Data Fetching
* Configuration and Environment Variables

## 🐛 Bug Reports
If you have found a bug in the library itself, please open a [GitHub Issue](../../issues/new?assignees=&labels=bug&template=bug_report.md&title=[BUG]+). Please provide a minimal reproducible example.

## 💡 Feature Requests
If you have an idea for a new feature, please open a [Feature Request Issue](../../issues/new?assignees=&labels=enhancement&template=feature_request.md&title=[FEATURE]+). We welcome suggestions!

## 💬 General Questions & Help
**Please do not open GitHub issues for general support or "how-to" questions.** 

For general React/WordPress integration questions, debugging help, or implementation advice, please use:
* **Stack Overflow:** Tag your question with `reactjs`, `wordpress-rest-api`, and `wp-react`.

## 🔒 Security Vulnerabilities
If you have found a security vulnerability, please **DO NOT** open a public issue. Refer to our [Security Policy](.github/SECURITY.md) for instructions on how to report it privately.
